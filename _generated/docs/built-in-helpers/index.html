Built in helpers

Add

Adds two numbers together

{{add @index 2}} <<< outputs @index + 2

can take multiple arguments

{{add 1 2 3}} outputs 6

Class

Render variable name if variable is true

{{class 'gradient'}} <<< will render 'gradient' if class context.gradient is truthy

can take multiple arguments

{{class 'gradient' 'dark' 'border'}}

Note: converts underscores to dashes

Compare

Compares two values and prints out one of two strings based on outcome of the comparison

{{compare age 20 'this dude is exactly 20 years old' 'hes not 20 years old}}

Default

Prints something in case that the provided variable is falsy

{{default age 20}} <<< if no age is provided 20 will be used
